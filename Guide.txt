GitHub Pages Step-by-Step Guide

Step 1: GitHub Login
- Open https://github.com
- Click Sign In
- Enter your username/email and password

Step 2: Create New Repository
- Click '+' icon → New repository
- Repository name: yourusername.github.io
- Select Public → Click Create repository

Step 3: Upload Website
- Open repository → Add file → Upload files
- Upload index.html (from the ZIP provided earlier)
- Click Commit changes

Step 4: Enable GitHub Pages
- Go to Settings → Pages
- Under Source → Select 'main branch'
- Click Save

Step 5: Access Website
- Wait 1-2 minutes
- Open https://yourusername.github.io
- Your website is live!
